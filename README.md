# scala-web-gradle-play-secure-3des-encryption-argon2-encode

## Description
A call to playframework web
with simple login forms argon2 hash
with 3des encryption.

## Tech stack
- scala
- gradle
  - play
  - 3des
  - argon2

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`
http://localhost
- username: user
- password: pass

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
